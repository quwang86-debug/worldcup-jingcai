## 一、区块链（BaaS）费用

国内公有云区块链服务（BaaS）主要面向**联盟链 / 企业链**场景，计费模式以**包年包月 / 按节点**为主，成本通常在**万元级 / 月**起。

### 1.1 阿里云区块链 BaaS

| 产品 / 引擎 | 规格 | 公开价格 |
| --- | --- | --- |
| **蚂蚁链 BaaS 专业版** | 4 节点（每节点 4核8G / 500GB） | **10 万元 / 年** |
| **蚂蚁链 BaaS 企业版** | 4 生产节点 + 1 测试节点（每节点 12核24G / 6TB） | **4.9 万元 / 月** 或 **50 万元 / 年** |
| **Hyperledger Fabric 联盟实例 · 基础版** | 100GB 免费存储 | **9,999 元 / 月** |
| **Hyperledger Fabric 联盟实例 · 企业版** | 500GB 免费存储 | **39,699 元 / 月** |
| **Hyperledger Fabric 联盟实例 · 企业安全版** | 1000GB 免费存储 | **59,699 元 / 月** |
| **Hyperledger Fabric 组织实例 · 基础版** | 100GB 免费存储 | **4,999 元 / 月** |
| **Hyperledger Fabric 组织实例 · 企业版** | 500GB 免费存储 | **22,699 元 / 月** |
| **Hyperledger Fabric 组织实例 · 企业安全版** | 1000GB 免费存储 | **39,699 元 / 月** |
| **超额存储** | 超出免费额度部分 | NAS 约 **1.85 元 / GB / 月**（按小时结算） |

**架构说明**：一个 Fabric 网络 = **1 个联盟实例 + N 个组织实例**，总费用随参与方数量叠加。

**官方文档**：[产品规格配置与计费定价规则](https://help.aliyun.com/zh/document_detail/107710.html) · [产品页](https://www.aliyun.com/product/baas)

### 1.2 腾讯云区块链 TBaaS（长安链 ChainMaker 标准版）

按 **节点数 × 节点规格** 计费（北京地域，包年包月）：

| 节点配置 | 单价 |
| --- | --- |
| 4核 CPU / 8GB 内存 / 200GB 磁盘 | **2,000 元 / 节点 / 月** |
| 8核 CPU / 16GB 内存 / 500GB 磁盘 | **2,938 元 / 节点 / 月** |
| 16核 CPU / 32GB 内存 / 1TB 磁盘 | **3,988 元 / 节点 / 月** |

**计费示例**（2 共识节点 + 2 同步节点，8核16G 配置，购买 2 个月）：

```
2,938 元/节点/月 × 4 节点 × 2 月 = 23,504 元
```

**注意**：创建网络至少需 1 个共识节点；正式业务建议至少 4 个共识节点保障高可用。

**官方文档**：[定价与规格说明](https://cloud.tencent.com/document/product/663/60095)

## 二、人脸识别费用

人脸识别服务分两大类，价格差异可达 **100 倍以上**：

| 类型 | 典型场景 | 价格区间 |
| --- | --- | --- |
| **纯 API** | 人脸检测、1:1 比对、1:N 搜索 | **0.0005 ~ 0.003 元 / 次** |
| **实名核身 / KYC** | 活体检测 + 身份证 / 权威库核验 | **0.15 ~ 1.5 元 / 次** |

---

### 2.1 纯人脸识别 API（低成本）

#### 阿里云 · 视觉智能开放平台（VIAPI）

| 能力 | 按量价格（阶梯） |
| --- | --- |
| **人脸比对 1:1** | 0.0016 元/次（≤50 万/月）→ 最低 **0.0008 元/次**（>5000 万/月） |
| **人脸活体检测** | 约 **0.0015 元 / Task**（按图片任务数计费，多图累加） |
| **人脸检测等** | 部分能力仍在公测，可免费使用 |

**官方文档**：[人脸人体费用](https://help.aliyun.com/zh/viapi/product-overview/billing-is-introduced-3)

## 三、横向对比与选型建议

### 3.1 区块链选型

| 场景 | 推荐方向 | 预估月成本 |
| --- | --- | --- |
| POC / 溯源试点 | 腾讯云长安链，4 节点 | **8,000 ~ 12,000 元 / 月** |
| 金融级合规 | 阿里云蚂蚁链专业版 | **≈8,300 元 / 月**（10 万/年） |
| 多组织联盟 | 阿里云 / 华为云 Fabric 架构 | **数万 ~ 数十万 / 月** |

### 3.2 人脸识别选型

| 场景 | 推荐方向 | 预估成本 |
| --- | --- | --- |
| 门禁、考勤、1:1 比对 | 纯 API（阿里 / 腾讯） | 百万次调用约 **数百 ~ 数千元** |
| 开户、实名、金融 KYC | 核身方案（百度性价比较高） | 每次 **0.15 ~ 1.5 元** |

### 3.3 成本估算示例

**示例 A：月调用 100 万次人脸比对（纯 API）**

| 厂商 | 估算 |
| --- | --- |
| 阿里云 | 100 万 × 0.0014 元 ≈ **1,400 元 / 月** |
| 腾讯云 | (100 万 - 1 万免费) × 0.0032 元 ≈ **3,168 元 / 月** |

**示例 B：4 节点联盟链 POC（腾讯云长安链，8核16G）**

```
2,938 元/节点/月 × 4 节点 = 11,752 元 / 月
```

**示例 C：月 1 万次实名认证（权威库）**

| 厂商 | 估算 |
| --- | --- |
| 腾讯云基础版 | 1 万 × 1.0 元 = **10,000 元 / 月** |
| 百度智能云 | 1 万 × 0.72 元 ≈ **7,200 元 / 月**（按第二阶梯） |
| 华为云 IVS 三要素 | 1 万 × 0.9 元 = **9,000 元 / 月** |
